# Simpod “盲听插锚 - 智能热区 - 批量复盘”泛听AI方案

## 背景与目标

Simpod聚焦在英语泛听与播客学习场景，通过极简流畅的“盲听插锚——AI智能热区——批量复盘”主流程，将传统泛听中冗杂的中途暂停、手动文本圈选操作完全摒弃。产品致力于为高效学习者、播客内容学习用户，以及高阶语言使用者，打造一体式、无打断、标记与回看兼备的AI泛听引擎。项目规划Mobile-First PWA为核心形态，并为原生多端埋好接口与数据基础。

### 达成指标

* 3个月内MVP开发上线，200名活跃种子用户。

* 单用户日均泛听/热区复盘≥10次，主流程平均无干扰率高于80%。

* 首月付费转化率目标5%，数据全链支持API/插件拓展。

## 核心场景与用户行为

### 真实泛听者使用场景（Narrative）

* **高频场景**：用户在地铁、碎片时间打开Simpod，无需打断当前音频，仅通过一次“盲听插锚”，主流程即可自动捕捉盲点。

* **播客深度学习**：长播客泛听中，AI实时推荐高概率障碍“热区”，用户可随时插入锚点，次日批量无缝复盘。

* **自定义拓展**：习惯批量导入难点或复习数据的高阶用户，仅需调用接口即可完成定制化标记和热区生成。

## 主流程聚焦：“盲听插锚——AI智能热区——批量复盘”

### 1\. 盲听插锚体验

* 用户上传音频或播客，系统自动识别文本并进入盲听主界面。

* UI仅保留播放与“插锚”两个主控入口：

  * “自动导航”模式下，AI实时判定并标记疑似障碍点，无需用户干预。

  * “手动插锚”入口支持用户任意位置插入锚点，适配极少量特定需求。

### 2\. AI智能热区推荐

* 系统基于第一轮盲听交互数据，批量预测可能的高难热区并自动圈定边界。

* 用户仅需一次批量确认或略微调整边界即可，无需频繁暂停与圈选。

* 热区边界支持连续滑动微调，查词、一键添加笔记、动态合并/拆分热区。

### 3\. 批量复盘 & 数据反哺

* “盲区复盘”入口，以热区为单位批量展开内容，支持“逐区揭晓”、复习顺序调节。

* 每次查词、扩展学习与笔记被自动归档至本热区，通过闭环数据反哺，AI可在下次自动优化热区建议与边界配置。

* 所有操作实现退出即保存，无需额外确认或重复标记。

## 数据结构与API设计（聚焦锚点盲听与智能批量）

### 关键数据结构设计

* `anchor` 支持盲听插锚的自动/手动触发方式，并预留来源区分（如自动导航、多次重叠标记等）。

* `hotzone` 动态包含上下文片段、边界调整历史、查词与笔记快照，兼容未来多设备、多场景批量导入。

* 全量日志数据被归纳整理，为AI下一轮热区推荐和展现优化做强化学习。

### API 重点

* POST /audio/upload 音频上传，同时自动调用AI转写接口，流程无拼接等待。

* GET /anchor/list?audio_id=xx 拉取全部锚点，自动区分手动与自动来源。

* POST /anchor/add 添加新锚点，可由AI或用户触发，包含来源标记。

* GET /hotzone/suggest?audio_id=xx 拉取AI批量建议热区，按置信度与优先级排序，全量一次返回。

* PATCH /hotzone/update 支持边界滑动与微调，保存后即自动反馈至AI优化体系。

* POST /review/batch 一键发起所有热区的复盘任务，自动归档查词、笔记、调整操作。

* 全程采用Apple风格极简圆角、流畅过渡动画，播放区与热区高对比灰白黑主色调。

* 所有主操作按钮底部大圆角悬浮，单手触及，支持手势滑动或长按激活。

* 扩展笔记、查词等入口随热区浮现，可动态滑入不遮挡主内容。

* 支持随时退出与后台自动保存，无打断确认弹窗。
